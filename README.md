# Chip-8 Emulator

A Chip-8 emulator written in C.<br>
Graphics done with the [Raylib](https://www.raylib.com/) library.<br>

## Usage

`chip8 [ROM file]`

Simply pass in the Chip-8 ROM file you want to run.
